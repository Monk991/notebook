## [文件系统路径的别名](https://cn.vitejs.dev/config/shared-options.html#resolve-alias)

### 配置

- vite.config.ts

```ts
// ...
export default defineConfig({
  // ...
  resolve: {
    alias: {
      '@': '/src'
    }
  }
  // ...
})
```

- tsconfig.json

```json
{
  "compilerOptions": {
    // ...
    "baseUrl": ".",
    "paths": {
      "@/*": ["src/*"]
    }
  }
  // ...
}
```
