### 下载框架插件

#### Windows

* 执行 update-oops-plugin-framework.bat 下载框架源码插件
* 执行 update-oops-plugin-hot-update.bat 下载热更新插件（可选）
* 执行 update-oops-plugin-excel-to-json.bat 下载Excel转Json格式插件（可选）
